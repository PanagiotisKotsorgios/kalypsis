# Kalypsis · Πιλοτικά Δεδομένα (Test Dummy Data)

Πακέτο δοκιμαστικών δεδομένων που ένας νέος χρήστης μπορεί να χρησιμοποιήσει
για να «παίξει» με την πλατφόρμα χωρίς να ρισκάρει πραγματικά στοιχεία.
Όλα τα δεδομένα είναι φανταστικά — ΑΦΜ, IBAN, τηλέφωνα και email έχουν φτιαχτεί
για δοκιμαστική χρήση μόνο.

---

## ⚠️ Email — Μπορεί η πλατφόρμα να στείλει;

**Ναι**, το BackOffice μπορεί να στείλει email (μέσω Brevo) στους παρακάτω χρήστες:

| Ροή                                 | Πότε στέλνεται                                 |
| ----------------------------------- | ---------------------------------------------- |
| Password reset                      | Όταν κάποιος πατά «Ξέχασα τον κωδικό»          |
| Email verification code (2FA)       | Αν έχει ενεργοποιηθεί email-code login         |
| Καμπάνιες marketing (πακέτο CRM)    | Χειροκίνητη αποστολή από το γραφείο           |
| Ειδοποιήσεις ανανεώσεων (roadmap)   | Αυτόματα ανά συμβόλαιο                        |
| Απάντηση σε αίτημα υποστήριξης     | Ο SuperAdmin απαντά μέσω του inbox            |

Για αυτό το λόγο **όλα τα email σε αυτά τα CSV είναι `@example.com`**.
Αυτός ο τομέας είναι δεσμευμένος από το IETF (RFC 2606) και δεν παραδίδει
mail — ό,τι στείλει η πλατφόρμα «θα φύγει στο κενό» χωρίς να ενοχλήσει κανέναν.

Αν θέλετε να δοκιμάσετε αποστολή email πραγματικά, αλλάξτε ένα-δύο email
σε δικό σας (π.χ. `esasan+kalypsis-test@yourdomain.gr`) πριν εισάγετε.

## 🚀 Πώς να τα χρησιμοποιήσετε

### Επιλογή Α: Χειροκίνητη καταχώρηση (γρήγορο για 1-2 εγγραφές)

Ανοίξτε τα CSV σε Excel, δείτε τη μορφή, καταχωρήστε 2-3 εγγραφές
από κάθε αρχείο μέσα από τη σελίδα «Πελάτες», «Συνεργάτες», κτλ.

### Επιλογή Β: Μαζική εισαγωγή (για ολόκληρο το πακέτο)

- **Ασφαλιστικές**: /app/insurance-companies → «Νέα ασφαλιστική» για κάθε γραμμή.
- **Συνεργάτες**: /app/producers → φόρμα δημιουργίας.
- **Πελάτες**: /app/customers έχει export/import (κατέβασμα → επικόλληση στη
  φόρμα ή απευθείας εισαγωγή αν υπάρχει «Εισαγωγή CSV»).
- **Συμβόλαια**: εναλλακτικά χρησιμοποιήστε τις **Γέφυρες Εταιριών**
  (/app/carrier-bridges) — ανεβάστε το CSV της ERGO/Interlife και θα
  δημιουργηθούν αυτόματα συμβόλαια, χρεώσεις και προμήθειες.
- **Πινάκιο υπερπρομηθειών**: πηγαίνετε στη «Μαζική Καταχώρηση Υπερπρομηθειών»
  (κουμπί μέσα στις Υπερπρομήθειες), επικολλήστε τις γραμμές, ελέγξτε τα
  αυτόματα υπολογισμένα split και πατήστε Εισαγωγή.

## 🧾 Συμβάσεις που ακολουθούμε στα CSV

- Delimiter: **`;`** (semicolon) — για συμβατότητα με Excel-el (κόμμα = δεκαδικός).
- Encoding: **UTF-8 με BOM** ώστε τα ελληνικά να εμφανίζονται σωστά στο Excel.
- Ημερομηνίες: `YYYY-MM-DD` (ISO 8601) — δεν εξαρτώνται από locale.
- Νούμερα: δεκαδικός με κόμμα (`1.234,56`) — Excel-el compatible.
- Email: πάντα `*@example.com` (μη-παραδοτέο).
